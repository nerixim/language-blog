# ロシア語名詞の格変化

---

## 1. 基本用語と略語

### 格変化と格

- **склоне́ние**：名詞の活用型  
- **падéж**：格

ロシア語の名詞は **語幹 + 語尾** から成り、文中での関係に応じて語尾が変化する。  
この体系を **склоне́ние**、変化後の各形を **падéж** と呼ぶ。

---

### 格の名称（6格）

| 略語 | 名称 |
|---|---|
| **И.п.** | Имени́тельный падéж |
| **Р.п.** | Роди́тельный падéж |
| **Д.п.** | Да́тельный падéж |
| **В.п.** | Вини́тельный падéж |
| **Т.п.** | Твори́тельный падéж |
| **П.п.** | Предло́жный падéж |

---

### 性・数の略語

- **м.р.** — мужско́й род  
- **ж.р.** — же́нский род  
- **ср.р.** — сре́дний род  
- **ед.ч.** — еди́нственное число  
- **мн.ч.** — мно́жественное число

---

## 2. 名詞の活用型（Ти́пы склоне́ния）

名詞の **И.п. ед.ч. の語尾と性** によって以下の3種類に分かれる。

| 活用 | 性 | И.п. ед.ч. |
|---|---|---|
| **I (пéрвое) склоне́ние** | ж.р., м.р. | -а / -я |
| **II (вторóе) склоне́ние** | м.р., ср.р. | -о / -е / -_ |
| **III (трéтье) склоне́ние** | ж.р. | -_（-ь） |

---

## 3. I склоне́ние, еди́нственное числó

### 概要

- **-а / -я** で終わる名詞
- 女性名詞が中心
- 一部の男性名詞も含む

### -ия 型の注意

**-ия** で終わる名詞は **Р.п. / Д.п. / П.п. が -и** になる。

例：информа́ция（情報）

| падéж | фóрма |
|---|---|
| И.п. | информа́ция |
| Р.п. | информа́ции |

| П.п. | об информа́ции |

## 9. 活用型の判定手順（要点）

1. 性（род）を確認
2. И.п. ед.ч. の語尾を見る
3. 以下で判断

- ж.р. / м.р. + **-а / -я** → I склоне́ние
- м.р. / ср.р. + **-о / -е / -_** → II склоне́ние
- ж.р. + **-ь / -_** → III склоне́ние
- ただし、例外がある

---

## 10. 活用早見表（ед.ч.）

| падéж | I скл. | II скл. | III скл. |
|---|---|---|---|
| И.п. | -а / -я | -о / -е / -_ | -_ |
| Р.п. | -ы / -и | -а / -я | -и |
| Д.п. | -е | -у / -ю | -и |
| В.п. | -у / -ю | -о / -е / -_ | -_ |
| Т.п. | -ой / -ей | -ом / -ем | -ю |
| П.п. | -е | -е | -и |

---

## 11. 活用早見表（мн.ч.）

| падéж | すべての склоне́ние |
|---|---|
| И.п. | -ы / -и / -а |
| Р.п. | -_, -ов, -ей |
| Д.п. | -ам / -ям |
| В.п. | И.п. または Р.п. |
| Т.п. | -ами / -ями |
| П.п. | -ах / -ях |

- мн.ч.（複数形）専用ページ
- 生物・無生物による В.п. の違い
- アクセント移動パターン
